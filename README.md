# 🌦️ AI Weather App

## 📌 About
A simple weather application that provides **real-time weather updates** and a **5-day forecast**, along with **AI-based clothing suggestions** based on weather conditions to help users plan their day.

---

##  Features
-  Real-time weather data
-  5-day forecast
-  Smart clothing suggestions
-  Clean and responsive design

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure
```
│── index.html
│── style.css
│── style.js
```

---

## ⚙️ How to Run
1. Download or clone the repository
2. Open the project folder
3. Double-click **index.html**  
   (or open it in any browser)

---
